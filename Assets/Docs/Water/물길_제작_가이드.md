# 물길 제작 가이드 (기획자용)

> **목표:** 청계천 같은 분위기 — *구불구불 흐르는 얕은 물 + 양쪽 돌둑/길*. 얕고 맑은 물, 가장자리 흰 거품.
> 코드 거의 안 만지고 깔 수 있게 정리함. (도구는 개발이 만들어둠)

---

## 0. 분위기 레퍼런스 (사진 기준)

| 요소 | 사진 속 | 우리 세팅 |
|---|---|---|
| 물 | 얕고 맑음, 가장자리 흰 거품 | Toon Water URP |
| 둑/길 | 회색 돌, 물이 살짝 가라앉음 | 회색 큐브 둑 + 수면 낮게 |
| 흐름 | S자로 구불구불 | 사인 곡선 물길 |

---

## 1. 30초 만에 보기 — 샘플 씬

상단 메뉴 **`Tools ▸ Sample ▸ Build Winding Water Scene`** 클릭
→ `Assets/Sample/Scenes/WindingWaterSample.unity` 가 **자동 생성되고 열림**.

보이는 것: 구불구불 물길 + 양쪽 둑. 이걸 그대로 참고용/복붙용으로 쓰면 됨.

---

## 2. 기획자 추천 — 슬라이더로 물길 깔기 (코드 X) ⭐

곡선·너비·둑을 **인스펙터 슬라이더로 실시간 조절**하는 컴포넌트가 있어.

1. Hierarchy 우클릭 ▸ **3D Object ▸ 구불구불 물길 (Winding Water)** → `WindingWater` 오브젝트 생성
2. 인스펙터 슬라이더를 움직이면 **씬에서 물길이 바로 다시 그려짐**:
   - **굽이 폭(amplitude)** / **굽이 개수(waves)** / **길이(length)**
   - **물길 너비(halfWidth)** / **수면 높이(waterHeight)** / **둑 높이(bankHeight)** / **둑 두께**
3. 오브젝트를 **이동/회전/스케일**하면 물길 전체가 같이 따라감 → 원하는 자리에 배치
4. 물 머티리얼은 자동으로 Toon Water가 지정됨 (인스펙터에서 교체 가능)
5. 모양 안 바뀌면 인스펙터 맨 아래 **"물길 다시 그리기"** 버튼

> 생성물(물·둑)은 자동 관리돼서 따로 신경 안 써도 됨. **이 `WindingWater` 오브젝트만** 원하는 씬으로 옮기면 그대로 재현돼.

### 평면 하나만 깔고 싶을 때 (곡선 필요 없으면)
1. `GameObject ▸ 3D Object ▸ Plane` 생성
2. **`Toon Water Material 1`** 을 평면 위로 드래그&드롭 → 끝 (높이는 둑보다 살짝 낮게)

---

## 3. 구불구불 물길 만들기

### 3-1. 샘플 도구로 뽑아서 복붙 (추천)
1. **1번**으로 샘플 씬 생성
2. Hierarchy에서 **`WindingWater`** + **`Banks`** 두 개를 선택 → 복사(Ctrl+C)
3. 실제 레벨 씬(예: `AnswerAuthoring`) 열고 붙여넣기(Ctrl+V)
4. 위치/회전/스케일로 원하는 자리에 맞춤

### 3-2. 모양(곡선/크기) 바꾸기
👉 **그냥 2번 컴포넌트 슬라이더로 바꾸면 됨** (제일 쉬움). 아래 표는 슬라이더 이름과 같음.
샘플 *씬 생성 도구*의 기본 모양만 바꾸려면 `Assets/Sample/Editor/WindingWaterSampleBuilder.cs` 상단 상수 수정 후 메뉴 다시.

| 값 | 의미 | 기본 | 크게 하면 |
|---|---|---|---|
| `kAmp` | 좌우로 굽이치는 폭 | 6 | 더 꼬불꼬불 |
| `kWaves` | 굽이 개수 | 3 | 굽이 많아짐 |
| `kHalfW` | 물길 너비(반폭) | 1.8 | 물 넓어짐 |
| `kWaterY` | 수면 높이 | 0.25 | 물 차오름(둑 넘으면 ↓) |
| `kBankH` | 둑 높이 | 1.0 | 둑 높아짐 |
| `kLength` | 전체 길이 | 60 | 물길 길어짐 |

> ✅ 이 값들은 이미 **2번 컴포넌트의 슬라이더**로 빠져 있음 — 인스펙터에서 바로 조절.

## 4. 물 느낌 튜닝 — Toon Water 머티리얼

물 표면 색·거품은 **머티리얼**에서. 프로젝트에서 `Assets/ThirdParty/Toon Water URP/` 의
**`Toon Water Material 1` / `2` / `3`** 중 골라 평면에 입히고 인스펙터에서 조절.

인스펙터에 뜨는 주요 항목(이름은 대략):
- **얕은 물 색 / 깊은 물 색** — 청계천은 맑은 하늘색~연파랑
- **거품 색 / 거품 양·거리** — 가장자리·둑에 닿는 흰 거품
- **표면 노이즈 / 왜곡** — 찰랑이는 무늬, 흐르는 느낌
- **투명도** — 얕게 보이려면 좀 투명하게

> 🔁 **원본 보존:** 머티리얼 고치기 전에 **복제(Ctrl+D)** 해서 우리 것(`Water_Cheonggye` 등)으로 만들고 그걸 수정. 원본 3종은 남겨두기.
> ⚠️ **거품이 안 보이면** → URP 에셋에서 **Depth Texture 켜기** (Project Settings ▸ Graphics ▸ 현재 URP Asset ▸ Depth Texture ✔). Toon Water 거품/가장자리는 깊이로 그림.

## 5. 청계천 분위기 체크리스트

- [ ] 물 **얕게** (수면 낮게), 둑 **회색**으로 낮게
- [ ] 물은 살짝 **투명** + 가장자리 **흰 거품**
- [ ] (디테일) `CFXR Water Ripples` 프리팹 몇 개 띄워 잔물결, 블록으로 작은 **다리**
- [ ] 달팽이들이 둑 위를 걸어다니면 스케일 살아남 (둑·바닥에 콜라이더 있음)

---

## 6. 트러블슈팅

| 증상 | 원인 / 해결 |
|---|---|
| 물이 **분홍색** | 셰이더 컴파일/URP 호환 문제 → 개발에게 |
| 물이 **안 보임**(밑에서만 보임) | 메쉬 노멀 뒤집힘 → 개발에게 (도구로 재생성하면 정상) |
| **거품 없음** | URP **Depth Texture** OFF → 켜기 (위 4번) |
| 물이 **둑 위로 넘침** | `kWaterY` 낮추거나 `kBankH` 높이기 |
| 샘플 메뉴 **안 보임** | 스크립트 컴파일 에러 확인(Console) |
