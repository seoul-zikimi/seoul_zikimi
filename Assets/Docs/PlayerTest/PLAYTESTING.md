# 플레이어 플레이테스트 가이드

> 기획자용 — 개발 환경에서 직접 테스트하는 방법

---

## 테스트 1 — 멀티플레이 이동 · 뷰 테스트

두 명이 같은 PC에서 Host / Client로 접속해 이동과 카메라를 확인합니다.

### 1-1. 최초 1회 세팅 (설치)

1. Unity 상단 메뉴 → **Window → Package Manager**
2. 좌측 상단 **Packages: Unity Registry** 선택
3. 검색창에 `Multiplayer Play Mode` 입력
4. 목록에서 선택 → **Install** 클릭
5. 설치 완료 후 Unity 재시작

### 1-2. 가상 클라이언트 창 추가

1. Unity 상단 메뉴 → **Window → Multiplayer → Multiplayer Play Mode**
2. 패널 하단 **+ (Add Player)** 클릭 → `Virtual Player 1` 생성
3. `Virtual Player 1` 체크박스가 **활성화** 상태인지 확인

### 1-3. 테스트 실행

1. **Project 창** → `Assets/Scenes/BootstrapScene` 더블클릭으로 씬 열기
2. Unity 상단 **▶ Play** 버튼 클릭
   - 에디터 Game 창 = **Host 창**
   - 별도로 뜨는 창 = **Client 창**

3. **Host 창**에서 `Host` 버튼 클릭
   - 자동으로 GameScene으로 전환되며 플레이어 스폰
4. **Client 창**에서 `Client` 버튼 클릭
   - 두 번째 플레이어 스폰

> ⚠️ **주의 — 클라이언트 접속 전 반드시 Host 플레이어 이동 필요**
> 두 플레이어는 같은 위치에 스폰됩니다. 클라이언트가 접속하기 전에
> Host 창에서 `W A S D`로 플레이어를 먼저 이동시켜 주세요.
> 이동하지 않으면 두 콜라이더가 겹쳐 낑기는 현상이 발생합니다.

### 1-4. 조작 방법

| 조작 | 동작 |
|---|---|
| `W A S D` | 캐릭터 이동 |
| 마우스 **우클릭 누른 채로** 드래그 | 카메라 회전 |
| 마우스 **스크롤** | 줌 인 / 아웃 |

### 1-5. 확인 항목

- [ ] 각자 자기 화면에만 카메라가 붙어 있는가
- [ ] WASD로 이동 시 카메라 방향 기준으로 움직이는가
- [ ] 카메라를 회전하면 이동 방향도 따라 바뀌는가
- [ ] 상대방 화면에서도 내 캐릭터 위치가 동기화되어 보이는가
- [ ] 두 캐릭터가 충돌하면 서로 튕겨나는가

## 테스트 2 — 충돌 · 이동 이펙트 테스트 (유진테스트 씬)

캐릭터끼리 부딪혔을 때 나오는 이펙트와 이동 중 먼지 효과를 확인합니다.  
이 테스트는 **혼자** 실행할 수 있습니다.

### 2-1. 씬 열기

1. **Project 창** → `Assets/Scenes/` 에서 유진테스트 씬 더블클릭
2. **▶ Play** 버튼 클릭

### 2-2. UI 버튼 설명

화면 좌상단에 버튼 3개가 있습니다.

| 버튼 | 동작 |
|---|---|
| **생성** | 캐릭터 4개를 원형으로 소환 |
| **모이기** | 캐릭터들이 중앙으로 이동 → 충돌 발생 |
| **초기화** | 모든 캐릭터 제거 |

### 2-3. 이펙트 테스트 순서

1. **생성** 버튼 클릭 → 캐릭터 4개 소환 확인
2. **모이기** 버튼 클릭 → 캐릭터들이 중앙으로 이동
3. 충돌 순간 확인 항목:
   - [ ] 충돌 이펙트(노란 불꽃 등)가 충돌 지점에 나타나는가
   - [ ] 캐릭터가 통통 튀는 애니메이션이 재생되는가
   - [ ] 충돌 후 캐릭터가 바깥으로 밀려나는가
4. 이동 중 확인 항목:
   - [ ] 캐릭터가 움직이는 동안 발밑에 먼지 이펙트가 나오는가
   - [ ] 멈추면 먼지가 사라지는가
5. **초기화** 버튼으로 정리 후 반복 테스트 가능

### 2-4. 이펙트 종류 변경 방법 (기획 조정)

이펙트 종류와 세기는 `PlayerConfig.asset`에서 조정할 수 있습니다.

1. **Project 창** → `Assets/Player/Data/PlayerConfig` 클릭
2. **Inspector**에서 수치 조정:

| 항목 | 설명 |
|---|---|
| **Bounce Effect** | 충돌 이펙트 종류 선택 (HitDYellow / HitMiscA / HitMiscFSmoke / Boing) |
| **Bounce Effect Duration** | 이펙트 지속 시간 (초) |
| **Rebound Force** | 충돌 후 밀려나는 힘 세기 |
| **Bounce Height** | 튀어오르는 높이 |
| **Bounce Duration** | 튀는 애니메이션 지속 시간 |
| **Move Speed** | 이동 속도 |
| **Dust Size** | 먼지 이펙트 크기 |

> **Tip:** Play 중에도 수치를 바꾸면 바로 반영됩니다.
